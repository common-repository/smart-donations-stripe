== Description ==

This plugin will allow you to create stripe donation buttons, you can learn more about smart donations here https://wordpress.org/plugins/smart-donations


== Installation ==

1. Upload smart donations (https://wordpress.org/plugins/smart-donations/) and smart donations stripe to the `/wp-content/plugins/` directory
2. Activate both plugins
